# vue-d3-force
  使用d3的力导向图模拟neo4j关系图谱展示<br>
  主要实现功能如下：<br>
      1、点击节点弹出圆环分区操作功能（包括解除节点位置锁定、删除节点和关系、隐藏节点和关系）<br>
      2、拖动节点后节点锁定位置功能<br>
      3、窗口拖动和zoom功能<br>
      4、点击连接线删除关系功能<br>
      5、绘制节点自己指向自己的关系连接线<br>
      6、两个节点多条关系连接线弧线绘制不重合功能<br>
      7、关系图谱容器自适应屏幕大小功能<br>
